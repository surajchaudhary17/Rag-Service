1. Download and install openjdk from eclipse
 link: https://adoptium.net/temurin/releases
2. Install Maven (Build tool)
👉 Download and Use Apache Maven
 Link: https://maven.apache.org/download.cgi   (download zip, -bin.zip)
 setups: Extract the ZIP to the path : "C:\maven" otherwise get error of too long path while extraction.
        Add bin folder to PATH : example C:\maven\apache-maven-3.9.15\bin
        Verify: mvn -version
        Note: do not download -src or mvnd (daemon)
	 ❌ source code (-src.zip)
            instead of
         ✅ binary (-bin.zip)

 
3. Install Docker Desktop
  Download from here (AMD for intel processor): https://www.docker.com/products/docker-desktop/
  It will take time: 
  - wsl stand for Windows Subsystem for Linux
  - Use powershell with admin permission and 
  - You may also required to update or install "wsl --install"and check status "wsl --status"
  - also required restart

4. Spring boot setup
   Website to make project: https://start.spring.io/ 

5. mvn clean install
6: host of spring boot: http://localhost:8080
7. to run spring boot server at 8080 port. command "mvn spring-boot:run" or you can just run using ui run icon
8. run this command : "docker compose up -d"
9. 🔹 Verify Redis
     command: "docker ps"
